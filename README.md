# Auth App

This is an auth app with login and registration functionality.

## Pages

1. Home Page
2. Register Page
3. Login Page
4. Profile Page

## Tech Stack

1. React + Vite
2. Tanstack Router
3. Tailwind CSS

## Getting Started

1. Clone the repository

2. Install dependencies

```bash
npm install
```

3. Copy .env.example to .env

```bash
cp .env.example .env
```

4. Add VITE_API_BASE_URL=https://api.freeapi.app/api/v1 to .env (optional)

5. Run the app

```bash
npm run dev
```
